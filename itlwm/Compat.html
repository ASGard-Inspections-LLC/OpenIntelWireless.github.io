# Compatibility

DANGER

NEVER use both itlwm.kext and Airportitlwm.kext at the same time!

Please use the card name displayed in Linux to match with the following names.

Specific PCI IDs can be found in ItlIwn.cpp (opens new window), mac80211.cpp (opens new window), and ItlIwx.cpp (opens new window).

Both AirportItlwm and itlwm support the following devices:
