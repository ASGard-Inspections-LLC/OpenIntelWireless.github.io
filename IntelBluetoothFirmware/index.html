# IntelBluetoothFirmware

IntelBluetoothFirmware is a project that provides Kexts for setting up the Bluetooth module to work natively in macOS.

Firmware blobs in this repository are from linux-firmware (opens new window).

  • IntelBluetoothFirmware.kext
  • IntelBluetoothInjector.kext
  • IntelBTPatcher.kext
